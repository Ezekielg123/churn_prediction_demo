# Customer Churn Prediction (Demo)

This project demonstrates a basic machine learning workflow for predicting customer churn
using a small, manually created dataset.

## Project Type
Learning / Demo Project

## Description
- Uses Logistic Regression
- Demonstrates data preparation, training, and evaluation
- Dataset is created manually for educational purposes

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## How to Run
```bash
pip install numpy pandas scikit-learn matplotlib
python churn_model.py
